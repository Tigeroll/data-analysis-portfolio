# 02 低回收率问卷的辅助信息估计工具

## 项目背景

在用户调研、满意度调查、A/B 测试小流量评估中，经常存在样本回收率低、样本代表性不足、指标波动大的问题。单纯依赖回收样本均值，可能导致结论不稳定；直接使用预测值，又可能把模型误差放大成业务结论。

本项目展示如何结合“全量辅助变量 + 预测值 + 少量真实观测”构建指标估计流程。

## 我的工作

1. 梳理低回收率问卷场景下的数据结构：全量用户特征、少量真实问卷结果、预测辅助信息；
2. 设计数据清洗规则，检查缺失、异常值、回收样本分布偏差；
3. 构建局部平滑 / 预测增强的估计思路，用真实观测修正预测偏差；
4. 进行参数敏感性分析，比较不同参数设置下的估计稳定性；
5. 输出可复用报告结构：数据口径、样本代表性、估计结果、风险提示和业务建议。

## 可迁移能力

- 低回收率问卷分析
- 辅助变量利用与样本代表性修正
- 参数敏感性分析
- 数据分析报告撰写
- 临时数据需求的快速响应模板
