# 03 文旅 IP 用户研究与商业分析

## 项目背景

围绕《黑神话：悟空》IP、山西文旅资源和数字藏品 / NFT 概念，本项目关注一个商业问题：

> 现象级 IP 带来的线上流量，能否转化为线下文旅消费和数字产品支付意愿？

## 我的工作

1. 设计用户研究框架，拆分游戏用户、文旅消费者和潜在游客；
2. 搭建“认知—兴趣—支付意愿—权益偏好—线下转化”的指标框架；
3. 完成问卷数据清洗、无效样本识别和基础描述统计；
4. 使用交叉分析和卡方检验识别支付意愿差异；
5. 将统计结果转化为产品和运营建议：权益分层、限量发行、线上线下联动、景区路线设计。

## 可迁移能力

- 问卷设计与用户研究
- 数据清洗与统计检验
- 用户分层与支付意愿分析
- 商业报告撰写
- 从数据结论到运营策略的转化
