# Data Analysis Portfolio｜李翔舟

本作品集用于展示我在数据收集整理、数据清洗、统计分析、机器学习建模、问卷研究和分析报告撰写方面的项目能力。项目均来自课程论文、科研训练、市调比赛和企业调研经历，并经过求职场景改写。

> 说明：部分原始数据受数据授权或隐私限制，仓库中提供的是脱敏样例数据 / 模拟数据，以及可复用的分析流程、代码模板和报告结构。

## 项目目录

| 项目 | 业务场景 | 展示能力 |
|---|---|---|
| 01 小样本调查数据校准与总体指标估计 | 标签少、辅助数据多时，如何稳定估计总体指标 | 数据清洗、特征工程、模型评估、偏差校正、指标解释 |
| 02 低回收率问卷的辅助信息估计工具 | 问卷回收率低或样本代表性不足时，如何提升估计稳定性 | Excel/Python 清洗、参数敏感性分析、报告模板、风险边界 |
| 03 文旅 IP 用户研究与商业分析 | 基于问卷数据判断用户支付意愿和产品策略 | 问卷设计、交叉分析、卡方检验、用户分层、策略建议 |

## 技术栈

- 数据处理：Excel, Python, pandas, numpy
- 数据分析：描述统计、交叉分析、卡方检验、回归分析、抽样估计
- 建模评估：机器学习预测、偏差 / MSE / 置信区间覆盖率、敏感性分析
- 报告输出：分析报告、可视化图表、业务建议、风险提示

## 如何查看

