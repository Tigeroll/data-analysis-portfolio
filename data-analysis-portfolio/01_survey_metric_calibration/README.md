# 01 小样本调查数据校准与总体指标估计

## 项目背景

企业常见场景中，经常存在“少量真实标签 + 大量辅助信息”的问题，例如：

- 用户满意度问卷回收少，但用户画像、行为记录较多；
- 人工审核样本成本高，但可获得大量基础特征；
- 收入、消费能力、留存价值等标签难以完整观测。

本项目目标是：在真实标签有限的情况下，利用辅助数据和预测模型提升总体指标估计的稳定性，同时避免直接使用模型预测造成系统性偏差。

## 我的工作

1. 整理多期样例数据，统一字段口径、缺失值处理和样本筛选规则；
2. 将行业、岗位、稳定性等类别变量整理为可建模字段，形成建模宽表；
3. 训练预测模型生成辅助预测值，并对“直接预测”和“校准估计”进行比较；
4. 评估偏差、均方误差、置信区间覆盖率等指标；
5. 将统计结果转化为业务语言：在标注预算有限时，如何降低指标误判风险。

## 可迁移能力

- 数据清洗与字段口径统一
- 高基数类别变量处理
- 预测模型评估与偏差检查
- 小样本估计、样本代表性修正
- 面向业务方输出分析报告与风险提示
